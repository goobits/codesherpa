# @goobits/sherpa

CLI guard and MCP setup for safer AI coding.

## Install

```bash
npx @goobits/sherpa init
```

Add your API key:

```bash
echo "CEREBRAS_API_KEY=..." >> .env
```

Restart Claude Code after running init.

## Commands

```bash
sherpa init
sherpa pre
sherpa post
sherpa daemon
sherpa status
```
